FEATURES
1. A main dashboard to view server information such as
  1. server uptime
  2. Servers currently active
  3. Player Counts
  4. Ram Usage
  5. CPU usage
  6. Available storage space
2. A Server Management page where users can start, stop and create new minecraft server instances.
3. A login system so that users can be given permissions to prevent user conflicts
4. an archive system to store unused servers on HD instead of SSD


FRAMEWORK STACK
1. Node will be used as the server backend
2. Express will host the server
3. EJS will be used to allow Javascript variable to be wrote inside of html so i can dynamically generate pages
4. SQL for database 
5. Docker will allow for easy deployment of the web server
6. Socket IO will be used to enable real time comunication between the server and clients
